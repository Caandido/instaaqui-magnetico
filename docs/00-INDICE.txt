 REGRA: TODA FEATURE FUTURA GANHA UM .TXT AQUI
------------------------------------------------------------------------------------------
  Sempre que uma nova fase ou feature for planejada/criada, deve ser adicionado um arquivo
  nesta pasta seguindo o MESMO modelo dos demais, com estas seções:

     1. Objetivo da fase/feature (em 1 parágrafo)
     2. O que esta etapa entrega (checklist)
     3. Pré-requisitos
     4. Passo a passo (com comandos)
     5. Como testar
     6. Arquivos criados/alterados
     7. Glossário dos termos novos

  Sugestão de nome para features futuras:
     FEATURE-<nome-curto>.txt   (ex.: FEATURE-EXPORTAR-PDF.txt, FEATURE-EQUIPE.txt)
